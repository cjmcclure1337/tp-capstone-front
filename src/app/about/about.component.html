About


This website was created to showcase the work done and skills developed as part of the Talent Path training program. This application allows you to select from a marketplace of stocks and foreign currency exchange (forex) and watch their value update in real time.

This website was built in Angular with some use of Angular Materials. It's supported by an ecosystem of back-end microserves built in Node.js utilizing the Express server structure. Data is stored in a mix of Postgres SQL and DynamoDB.

Design and implementation of the front end was done by Chris McClure. Chris is also reponsible for the Portfolio Management and Forex microservice servers on the back end. Other services, such as the Stocks Marketplace and the User Authentication service were designed by others as part of a larger Condor Financial project. The Condor Fiancial logo is from a design Lorc via Game-icons.net.
